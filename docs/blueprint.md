# **App Name**: Jalsa Hydration

## Core Features:

- Progress Circle Display: Display a circular progress bar visualizing current water intake against the daily goal.
- Quick Add Functionality: Implement buttons to quickly add +250ml (Glass) and +500ml (Bottle) to the intake.
- Custom Intake Input: Allow users to input a custom water intake amount.
- Goal Setting: Set a default daily water intake goal of 3000ml, which persists until the user changes it.
- Confetti Celebration: Trigger a confetti animation when the user reaches their daily water intake goal.
- Local Storage Persistence: Save the current water intake and goal to local storage to maintain data across sessions.
- Daily Reset Function: Implement a 'Reset for Today' button to reset the water intake to zero.

## Style Guidelines:

- Primary color: Soft, light blue (#ADD8E6) to evoke water and calmness.
- Background color: Very light desaturated blue (#F0F8FF).
- Accent color: Slightly darker desaturated blue (#A0B2B7) for buttons and interactive elements.
- Body and headline font: 'Inter', a sans-serif font for a clean, modern look.
- Use FontAwesome water-related icons (e.g., water drop) for visual cues.
- Mobile-first, centered layout with a soft shadow on a white card to hold the tracker elements.
- Use the canvas-confetti library to create a celebratory confetti explosion upon reaching the daily goal.
